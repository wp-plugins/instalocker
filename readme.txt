== Description ==

This is a Content Locker plugin which provides a shortcode to hide premium content from visitors until they follow you on Instagram. Once activated, you can add your Instagram username in InstaLocker Settings page and then use the shortcode [premium-content] to hide content from visitors until they follow you on Instagram.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `instalocker.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Instagram username in InstaLocker Plugin Settings page and verify your Instagram account.
4. Enclose the text which you want to hide inside `[premium-content] [/premium-content]` in your post.

== Frequently Asked Questions ==

= Why do I have to verify my Instagram account? =

We need to verify your Instagram account to make api calls on behalf of you.
